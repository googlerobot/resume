# 个人简历模板

## 介绍

由于找不到满意的简历模板，自己弄了一个，以满足自己的个性需求.

[在线预览](http://resume.f-tm.net)



## 特点

- [x] 简洁排版，支持响应式
- [x] 提供打印、下载PDF按钮及二维码
- [x] gulp自动构建
- [x] 自动发布到pages
- [ ] 自动生成pdf


## 命令

```
gulp serve 预览
gulp publish 发布到gh-pages分支

```
